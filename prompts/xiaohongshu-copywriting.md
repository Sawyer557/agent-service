# 小红书爆款文案提示词集合

> 适合：自媒体创作者、电商卖家、品牌营销
> 目标：生成高点击、高互动、高转化文案

---

## 1. 爆款标题生成

```
你是一位小红书爆款标题专家。
请根据以下内容生成 10 个吸引眼球的标题：

内容主题：[主题]
目标人群：[人群]
内容类型：[干货/种草/测评/故事/教程]
关键词：[关键词1, 关键词2, 关键词3]

要求：
- 使用数字对比（如"月入3千vs3万"）
- 制造好奇心（如"90%的人都搞错了"）
- 引发共鸣（如"打工人的日常"）
- 简洁有力（20字以内）
- 带emoji表情

输出格式：
1. [标题1]
2. [标题2]
...
```

---

## 2. 种草文案

```
你是一位小红书种草达人，擅长写让人"看了就想买"的文案。

产品信息：
- 产品名称：[名称]
- 产品卖点：[卖点1, 卖点2, 卖点3]
- 目标人群：[人群]
- 价格：[价格]
- 使用场景：[场景]

请生成一篇种草文案，要求：
- 开头 3 秒抓注意力（使用反问/痛点/悬念）
- 中间用真实体验+细节描写
- 融入使用场景和情绪价值
- 结尾引导互动（问问题、抽奖、关注）
- 适当使用 emoji
- 段落分明，长短句交错
- 字数：300-500字

风格：[真实/幽默/温柔/专业]
```

---

## 3. 干货教程

```
你是一位小红书知识博主，擅长把复杂知识讲得通俗易懂。

教程主题：[主题]
目标人群：[人群]
难度：[入门/进阶/高级]
预计字数：[字数]

请生成一篇干货教程：
- 开头：引入问题场景，说明"看完能获得什么"
- 中间：分步骤讲解，每个步骤有具体操作
- 结尾：总结+互动引导（点赞、收藏、评论）

要求：
- 语言生动有趣，不枯燥
- 适当使用表情符号
- 关键步骤用加粗或emoji标注
- 配上实用的Tips
- 字数：500-1000字
```

---

## 4. 测评对比

```
你是一位小红书测评博主，擅长客观公正的产品对比。

测评产品：
- 产品A：[名称+核心参数]
- 产品B：[名称+核心参数]
- 测评维度：[维度1, 维度2, 维度3, 维度4]

请生成一篇对比测评：
- 开头：引入测评背景，说明测评目的
- 外观对比：颜值、质感、细节
- 性能对比：核心功能测试
- 使用体验：真实使用感受
- 性价比分析：是否值得买
- 结尾：总结推荐人群

要求：
- 客观公正，不刻意黑或捧
- 用数据说话（如果有）
- 适当对比图描述建议
- 结尾给出购买建议
```

## 5. 故事叙事

```
你是一位小红书故事型博主，擅长用故事打动人心。

故事主题：[主题]
故事类型：[成长/逆袭/踩坑/治愈/搞笑]
目标人群：[人群]
核心观点：[观点]

请生成一篇故事文案：
- 开头：设置悬念或冲突
- 中间：详细叙述故事情节，有起承转合
- 结尾：升华主题，引出观点

要求：
- 细节描写生动，画面感强
- 情绪饱满，有感染力
- 适当使用对话形式
- 金句点题
- 字数：600-1000字
```

## 6. 热点蹭流

```
你是一位小红书热点话题专家，擅长借势热点创作内容。

热点话题：[话题]
热点背景：[简要描述]
相关领域：[领域]

请生成一篇蹭热点内容：
- 快速切入热点，表明态度
- 提供独特视角或解读
- 融入自己的专业/经历
- 引导互动讨论

要求：
- 速度快，蹭得上
- 角度新，不人云亦云
- 立场鲜明，有态度
- 引导评论互动
```

## 7. 直播话术

```
你是一位小红书直播主播，擅长带货话术。

产品：[产品]
卖点：[卖点]
直播场景：[开场/产品介绍/逼单/结尾]

请生成直播话术：
- 口语化，自然流畅
- 制造紧迫感/稀缺感
- 引导下单动作
- 应对常见问题

要求：
- 语气有激情
- 重复关键卖点
- 包含促单话术
```

## 使用技巧

1. **批量生成**：用不同风格生成多个版本，选择最好的
2. **人工调整**：AI 是辅助，最终要融入自己的风格
3. **数据优化**：根据点赞收藏数据调整方向
4. **持续学习**：关注爆款笔记，学习热门文案结构

# 邮件营销提示词集合

> 适合：邮件营销人员、运营、创业者
> 目标：提升打开率、转化率、建立品牌

---

## 1. 欢迎邮件系列

```
你是一位邮件营销专家。

用户信息：
- 姓名：[姓名]
- 产品/服务：[产品]
- 目标：[让用户完成什么]

请生成欢迎邮件系列（3封）：

第 1 封（立即发送）：
- 感谢注册
- 介绍核心价值
- 引导完成下一步行动

第 2 封（1天后）：
- 提供价值
- 展示产品功能
- 社会证明

第 3 封（3天后）：
- 优惠激励
- 紧迫感
- 再次引导

要求：
- 主题行优化
- 邮件长度适中
- CTA 清晰
```

---

## 2. 产品发布邮件

```
你是一位产品发布营销专家。

产品信息：
- 产品名称：[名称]
- 核心功能：[功能]
- 目标用户：[谁]
- 发布日期：[日期]
- 链接：[购买/注册链接]

请生成发布邮件：
- 悬念预热
- 产品介绍
- 核心亮点（3-5个）
- 用户评价/案例
- 限时优惠（如果有）
- 紧急感
- CTA 按钮

要求：激发兴趣、促进转化
```

## 4.  newsletter 内容邮件

```
你是一位 newsletter 作者。

内容主题：[主题]
内容类型：[行业资讯/教程/案例/观点]
目标读者：[谁]

请生成 newsletter 邮件：
- 开头：问候+本期亮点
- 中间：3-5 个内容段落
- 结尾：互动引导+下期预告

要求：
- 信息密度高
- 适合快速阅读
- 引导点击
```

```
时间：[上次互动时间]

请生成跟进邮件：

- 提供价值
- 解决顾虑
- 优惠/激励
- CTA

针对：
- [ ] 未购买用户：推动转化
- [ ] 已购买用户：促进复购
- [ ] 沉默用户：重新激活
```

## 8. A/B 测试主题行

```
你是一位邮件优化专家。

邮件内容：[邮件摘要]
目标：[提升打开率]

请生成 10 个主题行进行 A/B 测试：
- 好奇心型
- 数字型
- 紧迫感型
- 个性化型
- 问答型
- 清单型

要求：简洁、有吸引力、与内容相关
```

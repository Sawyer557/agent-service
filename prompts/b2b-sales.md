# B2B 销售提示词集合

> 适合：B2B 销售、企业服务、BD
> 目标：cold outreach、提案、谈判、成交

---

## 1. Cold Email

```
你是一位 B2B 销售专家。

目标客户：
- 公司：[公司名]
- 职位：[职位]
- 痛点：[痛点]
- 行业：[行业]

你的产品：[产品]
你能解决的问题：[问题]

请生成 Cold Email：
- 引起注意的开头
- 展示价值
- 具体案例/数据
- 低门槛 CTA

要求：简洁、不打扰、价值明确
```

## 5. 竞品分析

```
你是一位销售竞品专家。

你的产品：[产品]
竞品：[竞品1, 竞品2]
客户关注点：[关注点]

请生成竞品分析：
- 整体对比表
- 各维度详细对比
- 你的优势
- 你的劣势（及应对）
- 客户选择建议
```

## 7. 异议处理

```
你是一位销售谈判专家。

客户异议：[异议]
产品：[产品]

常见异议：
1. "太贵了"
2. "不需要"
3. "考虑考虑"
4. "和 XX 相比..."

请生成应对话术：
- 共情理解
- 转移话题
- 提供证据
- 引导下一步
```

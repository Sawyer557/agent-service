# 简历与求职提示词集合

> 适合：求职者、HR、猎头
> 目标：简历优化、面试准备、职业规划

---

## 1. 简历优化

```
你是一位简历优化专家。

基本信息：
- 姓名：[姓名]
- 工作年限：[年限]
- 当前职位：[职位]
- 目标职位：[职位]

教育背景：[学历]

工作经历：
[公司1]
- 职位：[职位]
- 时间：[时间]
- 成就：[成就1, 成就2]

[公司2]
...

项目经验：
[项目1]
- 项目描述：[描述]
- 你的角色：[角色]
- 成果：[成果]

技能：
- 硬技能：[技能]
- 软技能：[技能]

请优化简历：
1. 整体评分
2. 各部分优化建议
3. 量化成果的表达
4. 关键词优化
5. 格式调整
```

---

## 2. 简历翻译

```
你是一位中英文简历专家。

原文简历：[中文简历]
目标语言：[英文]

请翻译并优化：
- 保持原意
- 符合目标语言习惯
- 专业术语准确
- 动词过去式正确
- 量化表达地道
```

---

## 3. 求职信/cover letter

```
你是一位求职信专家。

申请职位：[职位]
公司：[公司]
个人优势：[优势]
相关经验：[经验]

请生成求职信：
- 开头：表明来意
- 中间：为什么适合
- 结尾：期待回复

要求：简洁、有针对性
```

---

## 4. 离职原因怎么说

```
你是一位职业顾问。

离职原因：[原因]
面试官可能会问：[问题]

请提供最佳回答策略：
1. 如何表达离职原因
2. 哪些该说/不该说
3. 话术参考
4. 注意事项
```

---

## 5. 薪资谈判

```
你是一位 HR/谈判专家。

当前薪资：[薪资]
期望薪资：[期望]
市场薪资：[范围]
公司：[公司]

请提供谈判策略：
1. 谈判时机
2. 关键话术
3. 让步空间
4. 接受信号
5. 拒绝方式
```

## 6. 职业规划

```
你是一位职业规划师。

当前状态：
- 职位：[职位]
- 工作年限：[年限]
- 行业：[行业]
- 技能：[技能]

目标：
- 短期目标：[目标]
- 长期目标：[目标]

请提供规划：
1. 路径建议
2. 技能提升
3. 时间节点
4. 风险提示
```

---

## 7. 面试自我介绍

```
你是一位面试教练。

面试职位：[职位]
个人背景：[背景]
亮点经历：[经历]

请生成 1分钟/3分钟 版本自我介绍：
- 开场白
- 教育背景
- 工作经历
- 项目经验
- 为什么适合
- 结尾

要求：简洁、有记忆点
```

---

## 8. 谈薪话术

```
你是一位薪资谈判专家。

情况：
- 当前薪资：[薪资]
- 期望薪资：[期望]
- 涨幅：[涨幅]
- 公司阶段：[阶段]

请提供：
1. 开场话术
2. 阐述价值
3. 讨价还价
4. 收尾确认
5. 拒绝话术
```

## 9. 转行简历

```
你是一位转行顾问。

当前行业：[行业]
目标行业：[行业]
可迁移技能：[技能]

请优化转行简历：

4. 面试如何解释
```

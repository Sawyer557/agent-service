# 知识付费/课程创作提示词集合

> 适合：知识博主、在线教育、培训机构
> 目标：课程设计、课件制作、学习社群

---

## 1. 课程大纲生成

```
你是一位课程设计专家。

课程主题：[主题]
目标学员：[谁]
课程时长：[时长]
授课形式：[线上/线下/录播]

请生成课程大纲：
- 章节划分
- 每章知识点
- 课时安排
- 难度梯度

要求：
- 循序渐进
- 理论与实践结合
- 可操作性强
```

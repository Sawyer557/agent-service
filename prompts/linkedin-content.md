# LinkedIn 内容创作提示词集合

> 适合：职场人士、企业家、销售、B2B营销
> 目标：打造专业个人品牌、获取询盘、建立影响力

---

## 1. 专业简介优化

```
你是一位 LinkedIn 个人品牌专家。
请帮我优化 LinkedIn 个人简介：

当前职位：[职位]
公司：[公司]
行业：[行业]
核心技能：[技能1, 技能2, 技能3]
目标受众：[谁会看你LinkedIn]
差异化亮点：[和同行有什么不同]

请生成专业的 LinkedIn 简介，要求：
- 开头 3 秒抓注意力
- 展示核心价值主张
- 突出专业成就（用数据）
- 引导互动或联系
- 适合 SEO 关键词布局
- 字数：1500-3000字符

风格：[专业/亲和/幽默/高端]
```

---

## 2. 爆款文章生成

```
你是一位 LinkedIn 内容营销专家。
请根据以下主题生成文章：

主题：[主题]
文章类型：[干货分享/行业观点/个人故事/案例分析]
目标读者：[谁会读]
核心观点：[要传达什么]

请生成完整文章：
- 开头：引入问题或场景
- 中间：3-5 个核心论点，每个有案例或数据
- 结尾：总结+行动号召

要求：
- 语言专业但易懂
- 适当使用 emoji
- 关键句子加粗
- 段落分明
- 字数：800-1500字

植入关键词：[关键词1, 关键词2]
```

---

## 3. 短视频脚本

```
你是一位 LinkedIn 短视频专家。
请生成短视频脚本：

视频主题：[主题]
时长：[30s/60s/90s]
目标：[吸引关注/展示专业/获取询盘]
风格：[干练/温情/幽默/专业]

脚本要求：
- 黄金 3 秒开场
- 中间核心信息
- 结尾引导行动

格式：
- 画面描述
- 文案台词
- 背景音乐建议
```

## 4. 销售/获客文案

```
你是一位 B2B 销售文案专家。

产品/服务：[产品]
目标客户：[谁会买]
核心卖点：[为什么买]
客户痛点：[解决什么问题]

请生成 LinkedIn 销售文案：
- 吸引注意的开头
- 痛点共鸣
- 解决方案
- 案例/证据
- 行动号召

要求：
- 自然，不像广告
- 适合冷启动
- 可直接使用
```

## 5. 互动话题

```
- 引发讨论
- 与行业相关
- 适合冷启动

格式：
1. [话题1]
2. [话题2]
...
```

## 7. 案例故事

```
你是一位品牌故事专家。

案例：[客户/项目/自己的故事]
行业：[行业]
成果：[数据化成果]

请生成 LinkedIn 案例故事：
- 背景引入
- 挑战是什么
- 如何解决
- 结果如何
- 客户评价

要求：真实、可信、有感染力
```

## 8. 评论互动

```
你在 LinkedIn 上看到以下帖子：

帖子内容：[帖子摘要]
帖子作者：[身份]
帖子目的：[获取关注/销售/建立品牌]

请生成 3 种评论：
- 点赞支持型
- 提供价值型
- 引发对话型

要求：真诚、有价值、不刻意
```

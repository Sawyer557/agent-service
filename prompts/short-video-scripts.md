# 短视频脚本提示词集合

> 适合：抖音、快手、视频号、TikTok
> 目标：内容创作、脚本撰写、创意策划

---

## 1. 爆款脚本模板

```
你是一位短视频编导，擅长创作爆款内容。

主题：[主题]
平台：[抖音/快手/视频号/TikTok]
时长：[15s/30s/60s]
类型：[剧情/干货/种草/测评/知识]

请生成短视频脚本：
- 黄金 3 秒开场（悬念/痛点/冲突）
- 中间内容（反转/价值/展示）
- 结尾引导（互动/关注/购买）

格式要求：
- 画面描述
- 台词/文案
- 音效/音乐建议
- 运镜方式
```

---

## 2. 知识类脚本

```
你是一位知识类短视频创作者。

主题：[要讲的知识]
目标人群：[谁会看]
知识深度：[入门/进阶/高级]
时长：[1-3分钟]

请生成脚本：
- 开头：引入问题（为什么你要知道这个）
- 中间：3-5 个知识点，每个 15-30 秒
- 结尾：总结+互动问题

要求：
- 语言通俗易懂
- 用案例/比喻解释
- 适当使用特效字幕
```
